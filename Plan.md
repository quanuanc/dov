# 构思

我想实现一个不同寻常的传输文件的工具，该工具可以通过 HDMI 来传输文件，具体做法是这样子：

1. A 运行sender，选定一个文件，然后将文件编码，以视频的方式显示在HDMI屏幕上
2. B 运行receiver，B电脑上接入一个采集卡，读取A输出的HDMI信号，然后就可以将视频信号解码恢复为文件了

# 实现过程

1. 先实现第一版 v1.0，该版本主要是为了测试协议，先以下面功能的方式实现，方便在同一台电脑上调试

## v1.0 功能

1. sender 通过读取指定文件，将其编码为视频
2. receiver 通过读取视频内容，将文件恢复出来，相当于从 sender 出接收到了文件

## v2.0 功能

1. sender 读取指定文件，输出到HDMI显示器上
2. receiver 从采集卡上读取视频信号，恢复为原始传输的文件

# 协议

视频的内容可以说是一种协议，该协议应该类似二维码，但是传输效率应该要高于二维码。
